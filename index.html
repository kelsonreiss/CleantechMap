Cleantech Market Map

A My Climate Journey Hackathon Project

We aim to build an interactive resource that visualizes the climate space in an intuitive, problem-oriented way. For example, we hope this will be a bridge between Project Drawdown's Solutions Table and a database of organizations working on those solutions.
